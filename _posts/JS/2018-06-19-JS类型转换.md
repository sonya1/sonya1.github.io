### 字符串与数字相加，会将数字转为字符串，然后拼接为字符串。
```js
2+"3"  //"23"
```

### 对象调用valueOf方法，把自己转换为数字，toString方法把自己转换为字符串

### 加法操作
布尔值参与“+”运算时，将其转为数字或字符串。
```js
8+true  //9
“8” + true  //“8true”
```

### 减法操作
两个操作数都会转为数字进行运算。

乘、除、大于、小于 和减法一样。

其他值类型进行比较时，都会先转成数字在比较。

## 
undefined == null  //true
